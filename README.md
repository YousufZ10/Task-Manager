# Task-Manager
Django Task Manager 

Task manager using Django python 

you can :

-Add Task

-Update Task

-Delete Task

-View All Tasks

-Set The Deadline For every Task

-Set Status ( pending / completed )

-Set Task Description 

-Set Task name and number based on your priority
